1. Créer une balise nav avec un id


                         
    <nav id="navcontainer">
        <ul> 
            <li class="active"><a href="#">Item one</a></li>
            <li><a href="#">Item two</a></li>
            <li><a href="#">Item three</a></li>
        </ul>
    </nav>
Explication 

Dans cette étape, une balise nav est créée dans laquelle id navcontainer est assigné pour appliquer un CSS spécifiques uniquement à cette balise nav.

2. Créer une liste avec un ul et des li contenant les liens href avec la balise a.

     
    <nav id="navcontainer">
        
        <ul>
            <li class="active"><a href="#">Item one</a></li>
            <li><a href="#">Item two</a></li>
            <li><a href="#">Item three</a></li>
        </ul>
        
    </nav>
Explication 

Dans cette étape une balise <ul> est crée avec les liste de balise <li> dans la quel on mets nos balise <a> pour pouvoir avoir un lien.

3. La première balise <li> est attribuer la classe active dans notre html.


         
      <nav id="navcontainer">
        <ul>
            
            <li class="active"><a href="#">Item one</a></li>
            
              <li><a href="#">Item two</a></li>
            <li><a href="#">Item three</a></li>
        </ul>
    </nav>
Explication 

La class active est attribué a la premier liste pour le premier affichage de nos menus comme active lien.

4. Code CSS du menu

                  
       
    #navcontainer ul {
        padding-left: 0;
        white-space: nowrap;
    }

    #navcontainer ul li {
        display: inline;
        list-style-type: none;
        
    }

    #navcontainer ul a {
        padding: 3px 10px;
    }
Explication 

Dans cet etape <ul> avec ID navcontainer on a mis sans padding et white-space: nowrap pour pouvoir mettre tous nos menus en une seule ligne.
Les <li> sont mis a display: inline pour s'afficher sur une ligne et sans puce avec l'option list-style-type: none.
Les balises <a> sont mis avec un padding haut et bas de 3px et gauche et droite de 10px.

5. Code CSS du block

     
        
    pre {
        margin: 0px 5px 15px 5px;
        background-color: var(--bgcol-pre);
        border: 1px #fff solid;
        border-radius: 10px;

Explication 

Dans cette etape les balise <code> est mis dans notre balise <pre> et a donne un margin haut de 0px, droit de 5px, bas de 15px et gauche de 5px.
Un bordeur pour notre block de code with un epessieur de 1px solid et en noir.
En suite les coins de notre bordeur est aroundi un peu a l'aide de border-radius.
Les paddings attribué sont haut de 5px, droit de 20px, bas de 5px et gauche de 5px.
